# Brent J. Marketing Car Parts Database

A Streamlit application for managing car parts inventory, clients, and generating quotes/invoices.

## Features

- Client management
- VIN number tracking
- Parts inventory with suppliers
- Quote and invoice generation
- PDF export functionality

## Installation

1. Clone this repository
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
